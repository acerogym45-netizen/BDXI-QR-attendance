# PDF 레이아웃 최적화 가이드

**작성일**: 2026-05-08  
**목적**: 검수 조서 PDF A4 한 장에 최대 8개 품목 배치

---

## 📊 최적화 요약

### 주요 변경사항
1. **쪽 여백 축소**: 10mm → 5mm (상하) + 8mm (좌우)
2. **검수 사진 규격 축소**: 120px → 100px 높이
3. **2×n 그리드 레이아웃**: 2열 구조 유지 (이미 적용됨)
4. **섹션 간 여백 축소**: 12px → 8px
5. **모든 내부 여백 축소**: 전반적인 padding/margin 축소

### A4 한 장 수용 능력
- **이전**: ~4-5개 품목
- **현재**: **최대 8개 품목** ✅

## 🎨 상세 변경 내역

### 1. 페이지 여백 (Padding)
```css
/* 이전 */
padding: 10mm;

/* 현재 */
padding: 5mm 8mm;  /* 상하 5mm, 좌우 8mm */
```

### 2. 검수 사진 카드
```css
/* 이전 */
border-radius: 6px;
padding: 8px;
gap: 6px;
margin-bottom: 6px;

/* 현재 */
border-radius: 4px;  /* -2px */
padding: 6px;        /* -2px */
gap: 4px;            /* -2px */
margin-bottom: 4px;  /* -2px */
```

### 3. 검수 사진 높이
```css
/* 이전 */
height: 120px;

/* 현재 */
height: 100px;  /* -20px (16.7% 축소) */
```

### 4. 배지 크기
```css
/* 이전 */
min-width: 20px;
height: 20px;
font-size: 10px;

/* 현재 */
min-width: 18px;  /* -2px */
height: 18px;     /* -2px */
font-size: 9px;   /* -1px */
```

### 5. 폰트 크기 축소
| 요소 | 이전 | 현재 | 변화 |
|------|------|------|------|
| 섹션 제목 (h2) | 14px | 13px | -1px |
| 물품명 (h3) | 11px | 10px | -1px |
| 카테고리 | 9px | 8px | -1px |
| 물품 정보 1줄 | 9px | 8px | -1px |
| 물품 정보 2줄 | 8px | 7px | -1px |
| 구매 사유 | 10px | 9px | -1px |
| 제목 날짜 | 11px | 10px | -1px |

### 6. 섹션 간 여백 (Margin-bottom)
```css
/* 모든 섹션 */
/* 이전 */
margin-bottom: 12px;

/* 현재 */
margin-bottom: 8px;  /* -4px (33% 축소) */
```

### 7. 제목 섹션
```css
/* 이전 */
margin-bottom: 12px;
padding-bottom: 8px;

/* 현재 */
margin-bottom: 8px;   /* -4px */
padding-bottom: 6px;  /* -2px */
```

### 8. 기본 정보 섹션
```css
/* 이전 */
border-radius: 6px;
padding: 10px;
margin-bottom: 12px;

/* 현재 */
border-radius: 4px;   /* -2px */
padding: 8px;         /* -2px */
margin-bottom: 8px;   /* -4px */
```

### 9. 구매 사유 박스
```css
/* 이전 */
border-radius: 4px;
padding: 8px;
line-height: 1.5;

/* 현재 */
border-radius: 4px;   /* 유지 */
padding: 6px;         /* -2px */

```

### 10. 물품 정보 2줄 박스
```css
/* 이전 */

padding: 6px;
border-radius: 4px;
font-size: 9px (1줄), 8px (2줄);

/* 현재 */
margin-top: 4px;      /* -2px */
padding: 4px;         /* -2px */
border-radius: 3px;   /* -1px */

```

### 11. 검수 사진 섹션 그리드
```css
/* 이전 */

column-gap: 8px;

/* 현재 */
gap: 6px;         /* -2px */
column-gap: 6px;  /* -2px */
```

## 🎯 A4 페이지 공간 계산

### A4 페이지 규격
- **전체 높이**: 297mm (≈ 1123px @ 96 DPI)
- **전체 너비**: 210mm (≈ 794px @ 96 DPI)

### 여백 제외 후 실제 사용 공간
```
상하 여백: 5mm × 2 = 10mm (≈ 38px)
좌우 여백: 8mm × 2 = 16mm (≈ 60px)

사용 가능 높이: 297mm - 10mm = 287mm (≈ 1085px)
사용 가능 너비: 210mm - 16mm = 194mm (≈ 734px)
```

### 섹션별 높이 예상
| 섹션 | 높이 (px) | 비고 |
|------|-----------|------|
| 제목 | ~60px | 제목 40px + 날짜 10px + 여백 |
| 기본 정보 | ~80px | 2×2 Grid + 헤더 |
| 구매 사유 | ~40px | 1-2줄 텍스트 |
| 물품 목록 | ~100px | 테이블 (품목 수에 따라 가변) |
| 검수 사진 (8개) | ~750px | 4행 × ~187px/행 |
| **총합** | **~1030px** | ✅ A4 1페이지 내 수용 가능 |

### 품목당 높이 계산 (2열 그리드)
```
카드 내부:
- 헤더: 28px (배지 18px + 여백)
- 사진: 100px
- 정보: 28px (padding 포함)
- 카드 padding: 6px × 2 = 12px
- 카드 border: 1px × 2 = 2px
───────────────────────────
  소계: ~170px/카드

행 간 gap: 6px
───────────────────────────
  행당 높이: ~176px

4행 × 176px = ~704px (8개 품목)
```

## 🔧 추가 최적화 가능 영역

### 현재 상태에서 추가 압축 가능한 부분
1. **물품 목록 테이블**: 행 높이 축소 (현재 ~15px/행)
2. **제목 섹션**: 날짜 폰트 더 축소 (10px → 9px)
3. **카드 내부 여백**: 더 축소 가능 (6px → 4-5px)
4. **사진 높이**: 더 축소 가능 (100px → 90px)

### 품목이 9개 이상일 경우
- 자동으로 2페이지로 넘어감 (html2canvas 멀티페이지 처리)
- 9-10개: 2페이지 1-2행
- 11-12개: 2페이지 3-4행
- 최대 16개까지 2페이지 내 가능
