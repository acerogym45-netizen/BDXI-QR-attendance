# 휴가 관리 시스템 빠른 시작 가이드 🚀

## 📍 어디서 찾을 수 있나요?

### 1단계: 근태 관리 탭 클릭
```
대시보드 → [근태 관리] 탭 버튼 클릭
```

### 2단계: 아래로 스크롤
```
캘린더 뷰를 지나서...
실시간 현황을 지나서...
근로시간 집계를 지나서...
초과근무 현황을 지나서...
지각/결근 현황을 지나서...
휴가 승인 대기를 지나서...
↓↓↓ 계속 스크롤 ↓↓↓
```

### 3단계: 휴가 관리 섹션 도착!
```
┌─────────────────────────────────┐
│ 🏖️ 휴가 관리                     │
│ 직원들의 휴가 신청 내역, 잔여...   │
└─────────────────────────────────┘
     ↑ 여기에 있습니다!
```

## 🖱️ 사용 방법

### 휴가 내역 보기
1. 테이블에서 모든 휴가 기록 확인
2. 필터로 원하는 항목만 표시

### 휴가 승인하기
1. 테이블에서 "대기중" 상태 찾기
2. ✅ 버튼 클릭
3. 확인 대화상자 → [확인] 클릭
4. 자동으로 "승인됨" 상태로 변경

### 휴가 거부하기
1. 테이블에서 "대기중" 상태 찾기
2. ❌ 버튼 클릭
3. 거부 사유 입력
4. [확인] 클릭
5. 자동으로 "거부됨" 상태로 변경

### 잔여 휴가 확인
1. 오른쪽 사이드바 → "직원별 잔여 휴가"
2. 각 직원의 진행률 바 확인
   - 🟢 초록색: 안전 (0-50% 사용)
   - 🟡 노란색: 주의 (50-80% 사용)
   - 🔴 빨간색: 경고 (80-100% 사용)

### Excel 다운로드
1. "Excel 다운로드" 버튼 클릭
2. 자동으로 CSV 파일 다운로드
3. Excel에서 열기
   - 한글 깨짐 없음 (BOM 포함)

## 🔍 필터 사용법

### 직원별 필터
```
[전체 직원 ▼] → [홍길동] 선택
→ 홍길동의 휴가만 표시
```

### 상태별 필터
```
[전체 상태 ▼] → [승인됨] 선택
→ 승인된 휴가만 표시
```

### 기간별 필터
```
[전체 기간 ▼] → [이번 달] 선택
→ 이번 달 휴가만 표시
```

### 복합 필터
```
[홍길동] + [승인됨] + [올해]
→ 홍길동의 올해 승인된 휴가만 표시
```

## 💡 팁 & 트릭

### 빠른 새로고침
- 상단의 [새로고침] 버튼 클릭
- 모든 데이터 자동 업데이트

### 휴가 정책 확인
- 오른쪽 사이드바 → "휴가 정책 안내"
- 연차/병가/경조사 규정 확인

### 신청 방법 안내
- 오른쪽 사이드바 → "휴가 신청 방법"
- 직원에게 안내할 4단계 프로세스

---

## ⚠️ 주의사항

### 필터 초기화
- 필터를 변경하면 자동으로 적용됨
- 전체 보기: 모든 필터를 "전체"로 선택

### 데이터 없을 때
- "휴가 기록이 없습니다" 메시지 표시
- 직원 앱에서 휴가 신청이 필요함

### 승인/거부 버튼
- "대기중" 상태에만 버튼 표시
- "승인됨" 또는 "거부됨"은 "-" 표시
- 승인/거부는 취소 불가능

## 🆘 문제 해결

### 문제: 휴가 섹션이 보이지 않음
**해결**: 근태 관리 탭에서 아래로 계속 스크롤

### 문제: 데이터가 로드되지 않음
**해결**: 브라우저 콘솔(F12) 확인 → 에러 메시지 확인

### 문제: 필터가 작동하지 않음
**해결**: 
1. 페이지 새로고침 (F5)
2. 테이블 상단의 [새로고침] 버튼 클릭

### 문제: Excel 파일 한글 깨짐
**해결**: 
- 이미 BOM 포함되어 있어 깨지지 않음
- Excel에서 "데이터 가져오기" 대신 직접 열기
