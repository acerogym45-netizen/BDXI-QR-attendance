# 구매 요청 Excel/PDF 추출 기능 테스트 가이드 🧪

## 🎯 테스트 목적

구매 요청 상세보기에서 **Excel**과 **PDF** 추출 기능이 정상적으로 작동하는지 확인합니다.

## 📋 사전 준비

### 1. 테스트 데이터 확인
- 구매 요청이 최소 1건 이상 등록되어 있어야 합니다
- 검수 사진이 업로드된 구매 건이 있으면 더 좋습니다

### 2. 접속 URL
```
https://bdxi-qr-attendance.vercel.app/index.html?apartment=[YOUR_APARTMENT_ID]
```

## 🔍 테스트 단계

### Step 1: 관리자 페이지 접속
1. 위 URL로 접속
2. 로그인 (아파트 ID 입력)
3. **"🛒 구매 요청"** 탭 클릭

### Step 2: 구매 상세보기 열기
1. 구매 요청 목록에서 아무 건이나 클릭
2. 상세 모달이 열리는지 확인

### Step 3: Excel 추출 버튼 확인
**예상 화면:**
```
┌─────────────────────────────────────────────┐
│  구매 요청 상세                  [X]         │
├─────────────────────────────────────────────┤
│  [기본 정보]                                │
│  [구매 사유]                                │
│  [물품 목록]                                │
│  [검수 사진 (있는 경우)]                    │
├─────────────────────────────────────────────┤
│  Actions:                                   │
│  [🟢 Excel] [🔴 PDF] [      닫기      ]     │
└─────────────────────────────────────────────┘
```

**확인 사항:**
- ✅ 초록색 "Excel" 버튼이 보이는가?
- ✅ 파일 아이콘(`fa-file-excel`)이 표시되는가?
- ✅ 버튼이 클릭 가능한 상태인가?

### Step 4: Excel 추출 테스트
1. **[Excel]** 버튼 클릭
2. 파일 다운로드 확인
   - 파일명: `구매요청_[요청자명]_YYYY-MM-DD.xlsx`
3. 다운로드된 파일 열기
4. **내용 검증:**
   - [ ] 기본 정보 (요청일, 상태, 요청자, 총 금액)
   - [ ] 구매 사유
   - [ ] 물품 목록 (물품명, 카테고리, 수량, 단가, 금액)
   - [ ] 검수 사진 정보 (물품명, 수량, 단가, 총액, 사진 URL)
   - [ ] 승인/반려 정보 (해당 시)

### Step 5: PDF 추출 버튼 확인
**확인 사항:**
- ✅ 빨간색 "PDF" 버튼이 보이는가?
- ✅ 파일 아이콘(`fa-file-pdf`)이 표시되는가?
- ✅ 버튼이 클릭 가능한 상태인가?

### Step 6: PDF 추출 테스트
1. **[PDF]** 버튼 클릭
2. 파일 다운로드 확인
   - 파일명: `구매요청_[요청자명]_YYYY-MM-DD.pdf`
3. 다운로드된 파일 열기
4. **내용 검증:**
   - [ ] 1페이지: 제목 "구매 요청서"
   - [ ] 1페이지: 기본 정보, 구매 사유, 물품 목록
   - [ ] 2페이지~: 검수 사진 (큰 이미지)
   - [ ] 각 사진 하단에 2줄 텍스트:
     - 1줄: `물품명: [물품명]`
     - 2줄: `수량: [N] | 단가: ₩[금액] | 총액: ₩[금액]`
   - [ ] 승인/반려 정보 (해당 시)
   - [ ] 페이지 넘김이 자연스러운가?

### Step 7: 이미지-물품 매칭 테스트
**조건:** 검수 사진이 있는 구매 건 선택

1. PDF 추출
2. 검수 사진 페이지 확인
3. **검증:**
   - [ ] 1번째 사진 아래 → 1번째 물품 정보
   - [ ] 2번째 사진 아래 → 2번째 물품 정보
   - [ ] 3번째 사진 아래 → 3번째 물품 정보
4. Excel 추출
5. **검증:**
   - [ ] "검수 사진" 섹션의 각 행이 사진 URL과 물품 정보를 포함

## 🐛 예상 오류 및 해결

### 오류 1: "구매 요청 데이터를 먼저 로드해주세요"
**원인:** 상세 모달을 열지 않고 버튼을 클릭
**해결:** 구매 건을 클릭하여 상세 모달을 먼저 연 후 버튼 클릭

### 오류 2: PDF 이미지 로드 실패
**원인:** CORS 문제 또는 이미지 URL 접근 불가
**해결:** 
- 브라우저 콘솔에서 오류 확인
- 이미지 URL을 직접 열어서 접근 가능한지 확인

### 오류 3: Excel 파일이 깨져 보임
**원인:** 한글 인코딩 문제
**해결:**
- Excel 대신 Google Sheets로 열기
- LibreOffice Calc로 열기

### 오류 4: PDF 파일이 깨져 보임
**원인:** 한글 폰트 미지원
**해결:**
- 현재는 Helvetica 폰트 사용
- 한글은 텍스트로 표시되나 폰트가 다를 수 있음
