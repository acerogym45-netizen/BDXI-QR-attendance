## 📐 A4 페이지 공간 활용

### 페이지 규격
- **A4 크기**: 210mm × 297mm (794px × 1123px @ 96 DPI)
- **여백 제외 후**: 194mm × 287mm (734px × 1085px)

### 섹션별 높이 분포
```
┌──────────────────────────────────┐
│ 제목 섹션          ~60px   (5%)  │
├──────────────────────────────────┤
│ 기본 정보          ~80px   (7%)  │
├──────────────────────────────────┤
│ 구매 사유          ~40px   (4%)  │
├──────────────────────────────────┤
│ 물품 목록          ~100px  (9%)  │
├──────────────────────────────────┤
│ 검수 사진 (8개)    ~750px  (69%) │
│   4행 × ~187px/행                │
├──────────────────────────────────┤
│ 여유 공간          ~55px   (5%)  │
└──────────────────────────────────┘
총 높이: ~1085px (A4 사용 가능 높이)
```

### 품목 카드 높이 계산
```
단일 카드 구성:
- 헤더 (배지 + 물품명): ~28px
- 검수 사진: 100px
- 물품 정보 2줄: ~28px
- 카드 padding (6px × 2): 12px
- 카드 border (1px × 2): 2px
─────────────────────────────
  카드 총 높이: ~170px

2열 그리드 행당 높이:
- 카드 높이: 170px
- 행 간 gap: 6px
─────────────────────────────
  행당 총 높이: ~176px

8개 품목 (4행):
  4행 × 176px = ~704px ✅
```
