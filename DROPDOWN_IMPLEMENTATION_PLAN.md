# 드롭다운 물품 선택 구현 계획

## 사용자 분석 (완벽히 동의)
- ✅ 카톡 파일명 (kakaotalk_xxx.jpg) → 파일명 매칭 불가
- ✅ 사진 각도/조명 문제 → AI 인식 불확실
- ✅ 순서 정렬 필요 → 번거로움 발생
- ✅ **드롭다운 선택이 가장 확실하고 간단**

## 구현 단계

### 1. DB 스키마 업데이트 (필수)
```sql
ALTER TABLE purchase_photos 
ADD COLUMN purchase_item_id UUID REFERENCES purchase_items(id);

CREATE INDEX idx_purchase_photos_item_id 
ON purchase_photos(purchase_item_id);
```

### 2. UI 개선
- 각 사진 미리보기 아래에 드롭다운 추가
- 물품 목록 표시: "김밥 (2개) - 간식"
- 선택 안 하면 업로드 시 경고

### 3. 업로드 로직 수정
- purchase_item_id 수집
- DB에 저장

### 4. 상세 페이지 개선
- 물품별로 그룹화 표시
- "김밥 (2개)" 헤더 → 해당 사진들
