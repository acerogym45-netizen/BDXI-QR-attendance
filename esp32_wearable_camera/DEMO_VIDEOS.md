## 🎨 영상 편집 (선택 사항)

### 자막 추가 방법

무료 영상 편집 도구:
- **Clipchamp** (Windows 11 내장)
- **iMovie** (macOS 내장)
- **CapCut** (모바일/PC)

#### CapCut으로 자막 추가:

```
1. CapCut 앱 다운로드 (무료)

2. "새 프로젝트" → 영상 업로드

3. 하단 "텍스트" 버튼

4. "자동 자막" 선택
   → 한국어 선택
   → 자동 생성

5. 수동 수정:
   0-3초: "기존 방식은 불편했습니다"
   3-6초: "손목에 차는 새로운 방식"
   6-10초: "버튼 한 번, 3초 완료!"

6. 내보내기 → 1080p → 저장
```

### 로고/워터마크 추가

```
회사 로고를 우측 하단에 추가:

1. 로고 이미지 준비 (PNG, 투명 배경)

2. CapCut에서 "스티커" → 로고 업로드

3. 크기 조절 (작게)

4. 위치: 우측 하단

5. 지속 시간: 전체 영상

6. 불투명도: 70%
```
