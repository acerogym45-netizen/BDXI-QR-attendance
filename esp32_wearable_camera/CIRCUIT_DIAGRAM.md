## 📐 핀 배치도

### ESP32-CAM 모듈 핀맵 (위에서 본 모습)

```
                    [WiFi 안테나]
        ┌───────────────────────────────┐
        │                               │
        │         ESP32-CAM             │
        │                               │
GND  ●──┤ 1                         16  ├──● 5V
IO12 ●──┤ 2                         15  ├──● 3V3
IO13 ●──┤ 3  [버튼 연결]           14  ├──● IO16
IO15 ●──┤ 4                         13  ├──● IO14
IO14 ●──┤ 5                         12  ├──● IO2
IO2  ●──┤ 6                         11  ├──● IO4 [내장 LED]
IO4  ●──┤ 7                         10  ├──● IO12
GND  ●──┤ 8                          9  ├──● GND
        │                               │
        │    U0R(RX) ●     U0T(TX) ●    │ [측면 핀]
        │       IO0 ●                   │
        └───────────────────────────────┘
                 [카메라 렌즈]
```

## 🔌 프로그래밍 연결도

### ESP32-CAM ↔ FTDI FT232RL (업로드 시)

```
        ESP32-CAM                    FTDI FT232RL
    ┌──────────────┐              ┌──────────────┐
    │              │              │              │
    │    5V     ●──┼──────────────┼──● VCC (5V)  │
    │    GND    ●──┼──────────────┼──● GND       │
    │    U0R    ●──┼──────────────┼──● TX        │
    │    U0T    ●──┼──────────────┼──● RX        │
    │              │              │              │
    │    IO0    ●──┼──┐           │              │
    │    GND    ●──┼──┘           │              │
    │              │   (업로드 시만)              │
    └──────────────┘              └──────────────┘
                                         │
                                    ┌────┴────┐
                                    │   USB   │
                                    │  Cable  │
                                    └────┬────┘
                                         │
                                    ┌────┴────┐
                                    │ Computer│
                                    └─────────┘

⚠️ 주의사항:
1. IO0을 GND에 연결해야 프로그래밍 모드 진입
2. 업로드 완료 후 IO0-GND 연결 제거
3. RX ↔ TX 크로스 연결 확인
```

## 🎛️ 버튼 회로 상세도

### GPIO 13 버튼 연결 (내부 풀업 사용)

```
       3.3V
         │
         │ (ESP32 내부 풀업 저항)
         │
         ├─────────● GPIO 13
         │
    ┌────┴────┐
    │  버튼    │
    │  ┌───┐  │
    │  │   │  │ ← 푸시 버튼
    │  └───┘  │
    └────┬────┘
         │
        GND

동작:
- 평상시: GPIO 13 = HIGH (풀업)
- 버튼 누름: GPIO 13 = LOW (GND 연결)
- 코드에서: pinMode(13, INPUT_PULLUP) 설정
```

### 외부 풀업 사용 (선택)

```
       3.3V
         │
       10kΩ (풀업 저항)

         ├─────────● GPIO 13
         │
    ┌────┴────┐
    │  버튼    │
    └────┬────┘

        GND
```

## 💡 LED 회로 상세도

### GPIO 33 LED 연결

```
                    ESP32-CAM
                  ┌──────────┐
                  │  GPIO 33 ●───┐
                  │          │   │
                  └──────────┘   │
                                 │
                              330Ω (전류 제한 저항)
                                 │
                              ┌──┴──┐
                              │ LED │
                              │  ●  │ ← 초록색 5mm LED
                              │  │  │
                              └──┬──┘
                                 │
                                GND

LED 극성:
  긴 다리 (+) → 저항 → GPIO 33
  짧은 다리 (-) → GND

전류 계산:
  I = (3.3V - 2.0V) / 330Ω ≈ 4mA
  (LED 순방향 전압: 2.0V)
```

```
    │                             │

    │   │  ─  18650 Cell   ─ │     │

    │         │         │         │
    │        (+)       (-)        │
    │         │         │         │
    │    ┌────┴────┐    │         │
    │    │ Switch  │    │         │
    │    │  ┌──┐   │    │         │
    │    │  │  │   │    │         │
    │    │  └──┘   │    │         │
    │    └────┬────┘    │         │
    │         │         │         │
    └─────────┼─────────┼─────────┘
              │         │
           (+) OUT   (-) OUT
              │         │
         ┌────┴─────────┴────┐
         │   ESP32-CAM        │
         │   5V          GND  │
         └────────────────────┘

전압: 3.7V (nominal) → 5V 부스터 필요 없음
      ESP32-CAM은 3.3V~5V 작동 가능

작동 시간: 약 8~12시간 (촬영 빈도에 따라)
```

## 🔧 배선 정리 팁

### 깔끔한 배선 방법

```
1. 와이어 길이 최소화
   ┌────┐   짧게 자르기
   │ESP │───● (5cm)
   └────┘

2. 열수축 튜브로 보호
   ─────●●●●●─────
        열수축

3. 글루건으로 고정
   ESP32-CAM
   ┌──────┐
   │  ●●  │ ← 글루건으로 와이어 고정
   └──────┘

4. 케이블 타이로 묶기
   ╔═══╗
   ║ ● ║ ← 케이블 타이
   ╚═══╝
```

```
     [카메라]
```
