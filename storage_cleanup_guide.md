# Supabase Storage 정리 가이드

## 1. Storage 메뉴 접속
- 왼쪽 사이드바에서 "Storage" 아이콘 클릭
- "cleaning-photos" 버킷 선택

## 2. 오래된 파일 삭제
- 날짜 정렬 (가장 오래된 순)
- 3개월 이상 된 사진 선택
- "Delete" 버튼 클릭

## 3. 대량 삭제 방법
- 폴더별로 선택 가능
- Shift + 클릭으로 여러 파일 선택
- 우클릭 → Delete

## 예상 효과
- 3개월 이상 된 사진 삭제 시: 약 1GB 절약
- 6개월 이상 된 사진 삭제 시: 약 1.3GB 절약
