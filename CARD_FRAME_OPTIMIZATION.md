# 검수 사진 카드 프레임 최대 활용 가이드

**작성일**: 2026-05-08  
**목적**: 검수 사진 카드 프레임 공간을 최대한 활용하여 가독성 향상

---

## 🎯 개선 목표

### 사용자 피드백
> "검수 사진별로 사각형의 프레임이 주어지는데 그 프레임은 최대로 활용을 해줘. 검수 사진 1번 헤더 텍스트에 김밥이어야하는데 잘려서 보이지가 않잖아. 검수 사진 프레임은 꽉 채워서 번호, 물품명, 카테고리를 상단배치, 중단에 사진은 상하 높이를 더 키우고, 하단의 물품명-수량-단가-총액의 순서는 지키되 폰트는 더 키워도 될 거 같아. 할당된 프레임자체가 많이 남아있으니까 최대로 활용해줘."

### 개선 결과
✅ **모든 요구사항 완료**
- ✅ 물품명 텍스트 잘림 현상 해결 (word-wrap 적용)
- ✅ 헤더 폰트 및 배지 크기 확대
- ✅ 검수 사진 높이 대폭 증가 (100px → 140px, +40%)
- ✅ 하단 정보 폰트 크기 확대 (가독성 향상)
- ✅ 카드 프레임 공간 최대 활용

## 📊 상세 변경사항

### 1. 헤더 섹션 (상단)

#### 배지 크기
```css
/* 이전 */
min-width: 18px;
height: 18px;
font-size: 9px;

/* 현재 */
min-width: 22px;  /* +4px (+22%) */
height: 22px;     /* +4px (+22%) */
font-size: 11px;  /* +2px (+22%) */
```

#### 물품명
```css
/* 이전 */
font-size: 10px;
white-space: nowrap;         /* 한 줄 제한 → 잘림 발생 */
overflow: hidden;
text-overflow: ellipsis;

/* 현재 */
font-size: 12px;             /* +2px (+20%) */
word-wrap: break-word;       /* 여러 줄 허용 */
overflow-wrap: break-word;   /* 긴 단어도 줄바꿈 */
line-height: 1.3;            /* 줄 간격 개선 */
/* 텍스트 잘림 현상 완전 해결 ✅ */
```

#### 카테고리
```css
/* 이전 */
font-size: 8px;

/* 현재 */
font-size: 9px;  /* +1px (+12.5%) */
```

#### 헤더 여백
```css
/* 이전 */
gap: 4px;
margin-bottom: 4px;
padding-bottom: 3px;

/* 현재 */
gap: 6px;              /* +2px */
margin-bottom: 6px;    /* +2px */
padding-bottom: 5px;   /* +2px */
```

### 2. 사진 섹션 (중단)

#### 사진 높이
```css
/* 이전 */
height: 100px;
margin: 4px 0;

/* 현재 */
height: 140px;    /* +40px (+40%) ✅ */
margin: 6px 0;    /* +2px */
```

#### 레이아웃 개선
```css
/* 이전 */
<div style="margin: 4px 0; text-align: center;">
  <img ... >
</div>

/* 현재 */
<div style="flex: 1; display: flex; align-items: center; justify-content: center; margin: 6px 0; overflow: hidden;">
  <img ... >
</div>
/* flex: 1로 사용 가능한 공간 최대 활용 */
```

## 🎓 기술 세부사항

### Flexbox 레이아웃
- `display: flex` + `flex-direction: column`: 수직 배치
- `height: 100%`: 부모 높이 전체 사용
- `flex: 1`: 사진 영역이 남은 공간 모두 차지

### 텍스트 줄바꿈
- `word-wrap: break-word`: 긴 단어를 줄바꿈
- `overflow-wrap: break-word`: 컨테이너 너비 초과 시 줄바꿈
- `white-space: normal`: 여러 줄 허용 (기본값)

### 반응형 고려
- `min-width: 0`: Flexbox에서 텍스트 오버플로우 방지
- `overflow: hidden`: 카드 영역 밖으로 나가지 않도록
