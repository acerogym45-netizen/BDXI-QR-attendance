# AC 작업 사진 촬영 시스템 - 사용 가이드

## 📸 두 가지 촬영 모드

### 1️⃣ 기본 모드 (Basic Mode)
**언제 사용하나요?**
- 구역을 하나씩 차례대로 작업할 때
- Before/After를 바로 연속해서 촬영할 때
- 진행 상황을 실시간으로 추적하고 싶을 때

**특징:**
- ✅ 구역 먼저 선택 → Before 촬영 → After 촬영
- ✅ 각 구역당 Before/After 1장씩
- ✅ 스마트 구역 추천 (다음에 할 구역 자동 제안)
- ✅ 전체 진행률 표시 (X/Y 완료, %로 표시)
- ✅ 완료 후 자동으로 다음 구역으로 이동

**사용 방법:**
1. "기본 모드" 버튼 클릭
2. 작업 구역 선택 (드롭다운)
3. "작업 전" 버튼 선택 → 사진 촬영
4. 자동으로 "작업 후"로 전환 → 사진 촬영
5. 구역 완료! 다음 추천 구역으로 자동 이동

---

### 2️⃣ 멀티 모드 (Multi Mode) 🆕
**언제 사용하나요?**
- 한 번에 여러 구역을 촬영한 후 나중에 정리할 때
- Before 사진을 모두 찍고, 작업 후 After를 한꺼번에 찍을 때
- 빠르게 많은 구역을 촬영하고 싶을 때

**특징:**
- ✅ 여러 장을 한 번에 선택 가능
- ✅ 각 사진마다 개별적으로 구역 지정
- ✅ Before/After 탭으로 구분 관리
- ✅ 사진 미리보기 + 삭제 가능
- ✅ 전체 업로드를 한 번에!

**사용 방법:**
1. "멀티 모드" 버튼 클릭
2. "작업 전" 탭 선택 상태에서 "사진 추가" 클릭
   - 여러 장을 한 번에 선택 가능
3. 각 사진마다 구역 선택 (드롭다운)
4. "작업 후" 탭으로 전환
5. "사진 추가" 클릭 → 작업 후 사진 선택
6. 각 사진마다 구역 선택
7. "전체 업로드" 버튼 클릭!

---

## 📊 비교표

| 특징 | 기본 모드 | 멀티 모드 |
|------|----------|----------|
| 구역 선택 | 먼저 선택 | 사진별로 나중에 선택 |
| 사진 수 | 각 1장씩 | 여러 장 가능 |
| 촬영 순서 | Before → After 순서대로 | 자유롭게 |
| 진행 추적 | 실시간 진행률 | 최종 업로드 시 |
| 사진 관리 | 즉시 업로드 | 검토 후 일괄 업로드 |
| 추천 구역 | ✅ 자동 추천 | ❌ 없음 |
| 사진 삭제 | ❌ 업로드 후 수정 필요 | ✅ 업로드 전 삭제 가능 |
| 적합한 상황 | 순서대로 작업 | 빠른 대량 촬영 |

## 💡 사용 팁

### 기본 모드 활용법
1. **스마트 추천 활용**: 노란색 박스에 표시되는 추천 구역을 따라가세요
2. **진행 현황 확인**: 상단의 진행률 바로 얼마나 했는지 확인
3. **자동 전환**: Before 촬영 후 자동으로 After로 전환되니 그냥 따라가세요

### 멀티 모드 활용법
1. **한 번에 선택**: 파일 선택 시 여러 장을 Ctrl/Cmd+클릭으로 한 번에!
2. **나중에 정리**: 급할 때는 일단 사진만 찍고, 나중에 여유 있을 때 구역 지정
3. **미리보기 활용**: 잘못 찍은 사진은 업로드 전에 삭제 가능
4. **탭 전환**: Before/After 탭을 자유롭게 전환하며 작업

## 🎯 어떤 모드를 선택해야 할까?

### 기본 모드 추천
- ✅ 처음 사용하는 경우
- ✅ 구역별로 차례대로 작업하는 경우
- ✅ 실시간 진행 상황을 보고 싶은 경우
- ✅ 헷갈리지 않게 단계별로 진행하고 싶은 경우

### 멀티 모드 추천
- ✅ 빠르게 많은 구역을 촬영해야 하는 경우
- ✅ Before 사진을 먼저 모두 찍고, 작업 후 After를 찍는 경우
- ✅ 사진을 검토하고 선택적으로 업로드하고 싶은 경우
- ✅ 여러 사람이 나눠서 촬영한 사진을 모아서 올리는 경우

## ❓ 자주 묻는 질문 (FAQ)

**Q: 두 모드를 섞어 쓸 수 있나요?**
A: 네! 상황에 따라 자유롭게 선택해서 사용하세요. 데이터는 동일하게 저장됩니다.

**Q: 사진을 잘못 올렸어요. 어떻게 삭제하나요?**
A: 관리자 페이지(index.html)의 "청소 작업 갤러리"에서 삭제 가능합니다.

**Q: 멀티 모드에서 구역을 선택 안 한 사진은?**
A: 업로드되지 않습니다. 구역을 반드시 선택해야 업로드됩니다.

**Q: 사진이 업로드되지 않아요.**
A: 인터넷 연결을 확인하고, 파일 크기가 너무 크지 않은지 확인하세요 (권장: 5MB 이하).

**Q: 진행 상황이 업데이트되지 않아요.**
A: 페이지를 새로고침하면 최신 데이터가 반영됩니다.
