# 🚀 빠른 시작 가이드: 카테고리 관리

## ⚡ 5분 안에 완료하기

### 1️⃣ 추가 카테고리 설치 (1분)

**Supabase SQL Editor로 이동**:
```
Supabase 대시보드 → SQL Editor → New Query
```

**다음 SQL 복사 & 실행**:
```sql
INSERT INTO document_categories (name, description, icon, color, display_order) VALUES
  ('안전관리', '안전 점검 보고서, 비상 대응 매뉴얼, 안전 교육 자료', 'fa-shield-alt', 'red', 6),
  ('시설관리', '시설물 유지보수 기록, 장비 점검 일지', 'fa-tools', 'orange', 7),
  ('법률', '법률 자문, 소송 관련 서류, 규정 문서', 'fa-gavel', 'indigo', 8),
  ('인사', '인사 규정, 근로계약서, 직원 평가서', 'fa-users', 'pink', 9),
  ('교육', '교육 자료, 매뉴얼, 교육 이수 기록', 'fa-graduation-cap', 'green', 10)
ON CONFLICT (name) DO NOTHING;
```

**"Run" 버튼 클릭** → ✅ Success!

### 2️⃣ 카테고리 관리 UI 사용 (2분)

#### 📋 카테고리 목록 보기
```
서류 관리 탭 → "📂 카테고리 관리" 버튼 클릭
```

#### ➕ 새 카테고리 추가
```
"새 카테고리 추가" 버튼 클릭
→ 정보 입력 (이름, 설명, 아이콘, 색상, 순서)
→ "저장" 버튼 클릭
```

#### ✏️ 카테고리 수정
```
카테고리 카드의 "✏️" 버튼 클릭
→ 정보 수정
→ "저장" 버튼 클릭
```

#### 🗑️ 카테고리 삭제
```
카테고리 카드의 "🗑️" 버튼 클릭
→ 확인
```
⚠️ **주의**: 서류가 있으면 삭제 불가

## 🆘 문제 해결

### Q: 카테고리가 표시되지 않아요
**A**: 페이지 새로고침 (F5) 또는 브라우저 캐시 삭제

### Q: SQL 스크립트 실행 시 오류 발생
**A**: 
1. RLS 정책이 활성화되어 있는지 확인
2. `document_categories` 테이블이 존재하는지 확인
3. 이미 동일한 이름의 카테고리가 있는지 확인

### Q: 카테고리 삭제가 안돼요
**A**: 해당 카테고리에 서류가 있는지 확인
- 서류 관리에서 해당 카테고리의 서류를 다른 카테고리로 이동하거나 삭제

### Q: 아이콘이 이상하게 표시돼요
**A**: 브라우저가 이모지를 지원하는지 확인
- Chrome, Firefox, Safari, Edge 최신 버전 권장
