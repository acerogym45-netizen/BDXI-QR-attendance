🔐 아파트별 비밀번호 컬럼 추가

📋 작업 내용:
apartments 테이블에 password 컬럼을 추가하여 각 아파트마다 개별 로그인 비밀번호를 설정할 수 있도록 합니다.
⚠️ 주의사항:
• 이 작업은 한 번만 실행하면 됩니다.
• 이미 컬럼이 있으면 에러가 발생할 수 있습니다 (정상).
• 기존 아파트는 기본 비밀번호 'bdxi2026'이 설정됩니다.

📝 Step 1: 컬럼 추가

ALTER TABLE apartments ADD COLUMN IF NOT EXISTS password TEXT DEFAULT 'bdxi2026';

📝 Step 2: 기존 데이터 업데이트

UPDATE apartments SET password = 'bdxi2026' WHERE password IS NULL;

✅ 마이그레이션 완료 후:

  1. 총괄 관리자 페이지에서 각 아파트의 비밀번호를 개별 설정할 수 있습니다.
  2. 로그인 시 선택한 아파트의 비밀번호로 인증됩니다.
  3. 기존 아파트는 계속 'bdxi2026'으로 로그인 가능합니다.
