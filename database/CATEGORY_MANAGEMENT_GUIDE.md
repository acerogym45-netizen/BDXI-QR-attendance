#### ✨ 새로운 기능

## 🚀 사용 방법

### A. 추가 카테고리를 데이터베이스에 삽입

#### 1단계: Supabase SQL Editor 열기
1. **Supabase 대시보드** 접속 (https://supabase.com)
2. 프로젝트 선택
3. 왼쪽 메뉴에서 **"SQL Editor"** 클릭

#### 2단계: SQL 스크립트 실행
1. **New Query** 버튼 클릭 (또는 `+` 아이콘)
2. 다음 파일 내용 복사: `/home/user/webapp/database/ADDITIONAL_CATEGORIES.sql`
3. SQL Editor에 붙여넣기
4. **"Run"** 버튼 클릭 (또는 `Ctrl/Cmd + Enter`)

#### 3단계: 결과 확인
성공 메시지와 함께 11개 카테고리가 조회됩니다:
```
✅ Success. No rows returned
```

그 다음 SELECT 쿼리 결과로 다음이 표시됩니다:
- 기존 6개 카테고리 (계약서, 공지문, 회계, 민원, 점검, 기타)
- 신규 5개 카테고리 (안전관리, 시설관리, 법률, 인사, 교육)

---

### B. 웹 애플리케이션에서 카테고리 관리

#### 카테고리 목록 보기
1. **서류 관리** 탭 클릭
2. **"📂 카테고리 관리"** 버튼 클릭
3. 등록된 모든 카테고리 확인

#### 새 카테고리 추가
1. 카테고리 관리 모달에서 **"➕ 새 카테고리 추가"** 버튼 클릭
2. 폼에서 정보 입력:
   - 이름 (필수)
   - 설명 (선택)
   - 아이콘 선택
   - 색상 선택
   - 정렬 순서 (숫자)
3. **"💾 저장"** 버튼 클릭
4. 성공 알림 확인

#### 카테고리 수정
1. 카테고리 목록에서 수정할 카테고리의 **"✏️ 수정"** 버튼 클릭
2. 폼에 기존 정보가 자동으로 채워짐
3. 원하는 정보 수정
4. **"💾 저장"** 버튼 클릭

#### 카테고리 삭제
1. 카테고리 목록에서 삭제할 카테고리의 **"🗑️ 삭제"** 버튼 클릭
2. 확인 대화상자에서 **"확인"** 클릭

⚠️ **주의**: 해당 카테고리에 서류가 있으면 삭제할 수 없습니다.
먼저 서류를 다른 카테고리로 이동하거나 삭제해야 합니다.
